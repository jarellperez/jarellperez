Detecting Land Change Using NASA's Earth Observations

Partnering with the USFWS, a user-friendly interface was developed in the Google Earth Engine API to display remotely sensed imagery obtained from Landsat 5 TM, Landsat 7 ETM+, Landsat 8 OLI, and Sentinel-2 MSI satellites. These data were used to remotely determine past land use changes in the HCPs from 2000 to 2017.
